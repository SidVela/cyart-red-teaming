# Threat Hunting Report

## Objective
Monitor Linux system logs using Filebeat, Elasticsearch, and Kibana.

## Tools Used
- Elasticsearch
- Kibana
- Filebeat
- Sigma Rules

## Activities Performed
- Installed and configured Filebeat
- Enabled system log collection
- Ingested Linux syslog and auth logs
- Visualized events in Kibana Discover
- Simulated suspicious PowerShell execution
- Simulated failed authentication attempts
- Simulated reconnaissance activity

## Results
Successfully detected suspicious events in Kibana.

## Outcome
Built a working centralized log monitoring and threat hunting environment.
